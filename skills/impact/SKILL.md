## Comment utiliser cette bibliothèque

1. **Auto-trigger** : ce skill se charge dès qu'une question touche à la formation, la communication, la pédagogie, les biais, l'enneagramme, etc.
2. **Fichiers référence** : chaque thème a son fichier `.md` dans ce répertoire. Lis le fichier correspondant pour obtenir les concepts complets.
3. **Orchestration** : si plusieurs thèmes sont concernés, combines les fichiers pertinents.

## Sélection selon la situation

**Améliorer un texte / email / argumentaire** → `ecriture.md` (orchestrateur) + `accroche.md` + `adapter.md`

**Concevoir une formation** → `structure.md` + `action.md` + `storytelling.md`

**Gérer la résistance au changement** → `emotion.md` + `persuasion.md` + `biais.md`

**Adapter à un profil psychologique** → `enneagramme.md` (lire le type concerné)

**Donner / recevoir un feedback** → `feedback.md` + `expression.md`

**Faciliter une réunion ou atelier** → `facilitation.md` + `questionnement.md`

**Trouver un titre ou nom** → `accroche.md`

**Présentation, pitch, slides** → `storytelling.md` + `structure.md`

**Creer ou reviser des quiz QCM** → `quiz-design.md` + `biais.md` + `adaptive-learning.md`

**Gamifier une formation / augmenter la complétion** → `gamification.md` + `biais.md`

**Concevoir un parcours adaptatif (Duolingo de l'IA)** → `adaptive-learning.md` + `gamification.md` + `instructional-design.md`

**Ingénierie pédagogique : de l'objectif au livrable** → `instructional-design.md` + `structure.md` + `adaptive-learning.md`

**Vision "Duolingo de l'IA" — formation adaptative complète** → `vision-duolingo-ia.md` + `gamification.md` + `adaptive-learning.md`

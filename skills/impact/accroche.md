# SKILL : impact-accroche

> Trouver en quelques mots ce qui fait tout — le titre, le slogan, la formule qu'on ne peut plus oublier.

---

## Quand activer ce skill

- "Trouve-moi un titre pour..."
- "Il me faut un nom pour cette formation / ce projet / ce slide"
- "Comment accrocher mon audience en 5 mots ?"
- "Mon titre est plat, ça manque de punch"
- Création de contenu : newsletter, présentation, post LinkedIn, atelier

---

## AC.1 — Pourquoi les titres font (presque) tout

**Fait empirique :** 80% des personnes lisent le titre, 20% lisent le reste.  
*(Copyblogger, étude sur la lecture web — repris par BuzzSumo et Nielsen Norman Group)*

Un titre n'est pas un résumé. C'est une **promesse**, une **invitation**, ou une **provocation**.

### Ce que fait un bon titre

```
1. Il capte l'attention (stop-scroll, stop-scan)
2. Il qualifie l'audience ("ce n'est pas pour toi" est un filtre utile)
3. Il donne envie d'aller plus loin
4. Il se retient (mémorisabilité)
5. Il fonctionne seul, hors contexte
```

### La hiérarchie des titres

| Force | Exemple | Mécanisme |
|-------|---------|-----------|
| Titre descriptif | "Introduction à Excel" | Informatif, zéro émotion |
| Titre bénéfice | "Excel pour gagner 2h par semaine" | Promet un résultat |
| Titre tension | "Vous utilisez Excel comme en 2005" | Crée une inconfort |
| Titre intrigant | "La formule qu'Excel ne veut pas que vous connaissiez" | Curiosité gap |
| Titre choc | "Excel est mort. Voici ce qui le remplace." | Provocation assumée |

## AC.2 — Les 6 types d'accroches

### 1. La Promesse de bénéfice
```
"Apprenez à négocier en 4 heures."
"Réduisez vos réunions de 30% dès lundi."
```
**Recette :** Verbe d'action + résultat mesurable + délai ou condition

### 2. La Question rhétorique
```
"Et si vous pouviez dire non sans vous justifier ?"
"Vous avez encore relu vos emails 3 fois avant d'envoyer ?"
```
**Recette :** Question qui met le doigt sur une frustration connue

### 3. Le Paradoxe / Oxymore
```
"Travailler moins pour en faire plus."
"L'art de l'échec productif."
"Simple. Profond. Durable."
```
**Recette :** Deux concepts opposés réconciliés → le cerveau cherche la résolution

### 4. Le Nombre précis
```
"7 mots pour captiver une salle."
"Les 3 erreurs que font 90% des managers."
"Formation en 12 modules, 0 jargon."
```
**Recette :** Chiffre impair → plus crédible qu'un chiffre rond · Chiffre précis → plus crédible que vague

### 5. L'Injonction directe (impératif)
```
"Stop aux réunions inutiles."
"Écris. Maintenant."
"Prends ta place."
```
**Recette :** Un verbe à l'impératif, court, sans fioritures

### 6. Le Contraste temporel
```
"Hier formateur. Demain facilitateur."
"Ce que vous faisiez en 3 jours. En 3 heures."
"Avant : subir le changement. Après : le piloter."
```
**Recette :** Avant/Après · Hier/Demain · Eux/Vous

## AC.3 — Figures de style pour titres — la boîte à outils

### Allitération
Même son au début de plusieurs mots. Rythme, mémorisation.
```
"Former, Fédérer, Faire grandir."
"Pitch, Persuasion, Performance."
"Simple. Sobre. Solide."
```

### Tricolon (règle de 3)
3 éléments en accumulation croissante. La troisième surprend.
```
"Veni, vidi, vici." (César)
"Voir, comprendre, agir."
"Rapide. Précis. Implacable."
```
*Le 3e élément doit être le plus fort — jamais anticlimax.*

### Chiasme
Structure croisée A B / B A. Élégant, paradoxal.
```
"On ne vit pas pour travailler, on travaille pour vivre."
"Ce n'est pas la formation qui fait le formateur — c'est le formateur qui fait la formation."
"Less is more / More is less."
```

### Anaphore
Répétition en début de phrases. Frappe comme un marteau.
```
"J'ai un rêve. J'ai un rêve..." (MLK)
"On mérite mieux. On attend mieux. On fait mieux."
```

### Assonance / Rime
Son vocalique répété. Rend mémorable même sans raisonner.
```
"Former pour transformer."
"Savoir-faire et savoir-être."
"Data, Agilité, Clarté."
```

### Ellipse (sous-entendu)
Omettre des mots — la phrase est plus forte.
```
"Moins de réunions. Plus d'impact." (sous-entendu : tu peux avoir les deux)
"Moins de blabla." (le reste est évident)
```

## AC.4 — La contrainte des 6 mots

Inspiré de la légende Hemingway (*"For sale: baby shoes, never worn"*) — la contrainte génère la créativité.

**Exercice :** Résumer une formation, un projet ou une idée en **6 mots maximum**.  
Pas de verbe "être", pas de "et", pas de "qui".

### Méthode

```
Étape 1 — Écrire la version longue (ce qu'on voudrait dire)
Étape 2 — Identifier le mot le plus fort → le garder
Étape 3 — Couper tout ce qui est implicite ou déductible
Étape 4 — Reformuler avec des noms forts et des verbes d'action
Étape 5 — Tester la version 6 mots : comprend-on encore le cœur ?
```

### Exemples

| Version longue | Version 6 mots |
|----------------|---------------|
| "Formation pour apprendre à mieux gérer son temps" | "Reprendre le contrôle de son agenda" |
| "Module sur la communication non-violente en équipe" | "Parler vrai. Travailler ensemble." |
| "Accompagnement des managers dans la conduite du changement" | "Managers : pilotez le changement, pas l'urgence." |

## AC.5 — Naming : formation, produit, projet, skill

Le naming est l'acte de trouver un **nom propre** qui devient une identité.

### Les 4 stratégies de naming

| Stratégie | Principe | Exemple |
|-----------|---------|---------|
| **Descriptif** | Nom = ce que c'est | "Formation Prise de parole" |
| **Métaphorique** | Nom = image évocatrice | "PitchLab", "Le Chantier", "Sprint" |
| **Acronyme signifiant** | Lettres initiales → mot | IMPACT, CLEAR, BOLD |
| **Néologisme** | Mot inventé, mémorable | "Bootcamp", "Masterclass", "Hackathon" |

### Test d'un bon nom

```
□ Se prononce-t-il facilement à l'oral ?
□ S'écrit-il sans hésitation ?
□ Fonctionne-t-il hors contexte (sans sous-titre) ?
□ N'existe-t-il pas déjà pour autre chose dans l'organisation ?
□ Évoque-t-il quelque chose de positif / neutre (pas d'homophone fâcheux) ?
□ En anglais aussi, si usage international ?
```

### Exercice : générer 10 variantes de nom

1. Nom purement descriptif
2. Nom avec un verbe d'action
3. Nom métaphorique (outil, lieu, voyage)
4. Nom avec un chiffre
5. Nom avec une allitération
6. Acronyme (chercher 5-6 lettres en lien avec le bénéfice)
7. Nom anglais / hybride
8. Nom en question
9. Nom provocateur / contre-intuitif
10. Nom minimaliste (1-2 mots)

→ Sélectionner les 3 meilleurs · Tester en contexte · Décider

## AC.6 — Slogan vs Baseline vs Tagline

Trois concepts souvent confondus, trois fonctions distinctes.

| Concept | Définition | Durée de vie | Exemple |
|---------|-----------|-------------|---------|
| **Tagline** | Positionnement de la marque / produit | Long terme | "Just Do It" (Nike) |
| **Baseline** | Complément du nom, sous le logo | Long terme | "Impossible is Nothing" (Adidas) |
| **Slogan** | Message de campagne, contexte précis | Court terme (campagne) | "Parce que vous le valez bien" (L'Oréal) |
| **Accroche** | Phrase d'entrée d'un contenu | Unique au contenu | Premier paragraphe d'un article |

### Pour les formations et projets internes

La distinction pertinente :
- **Titre** = le nom officiel de la formation / projet
- **Sous-titre** = la promesse (bénéfice en une ligne)
- **Description courte** = 25-50 mots, catalogue / email

```
Titre : "Prise de parole en public"
Sous-titre : "Captiver une salle en toutes circonstances"
Description : "2 jours pour structurer votre message, gérer le trac
et marquer les esprits — du one-to-one à la conférence de 300 personnes."
```

## AC.7 — Test de mémorabilité : 5 critères

Un titre ou slogan est mémorable si on peut le scorer positivement sur ces 5 dimensions.

| Critère | Question | Outil |
|---------|----------|-------|
| **Brièveté** | Moins de 7 mots ? | Compter |
| **Musicalité** | Se lit bien à voix haute ? | Lire à voix haute |
| **Unicité** | Pourrait appartenir à une autre organisation ? | Recherche rapide |
| **Clarté** | Compris instantanément sans contexte ? | Tester sur quelqu'un d'externe |
| **Désirabilité** | Donne-t-il envie de la suite ? | Réaction émotionnelle subjective |

## AC.8 — Anti-patterns des titres ratés

| Anti-pattern | Exemple | Pourquoi ça échoue |
|-------------|---------|-------------------|
| **Trop générique** | "Formation Management" | Ne différencie pas, n'intrigue pas |
| **Jargon interne** | "Module RPS-2026-V3" | Sens zéro hors organisation |
| **Liste de mots-clés** | "Communication, Leadership, Confiance" | Catalogue, pas invitation |
| **Promesse impossible** | "Devenez un expert en 1 jour" | Incrédulité immédiate |
| **Titre question + réponse** | "Comment communiquer ? Les bonnes pratiques" | Double structure, perd le focus |
| **Trop long** | "Formation aux techniques de communication non-violente pour managers en contexte de transformation" | Essoufflement avant la fin |
| **Verbe faible** | "Introduction à...", "Initiation à...", "Approche de..." | Minimise l'enjeu |

## AC.9 — Atelier : protocole de génération

Pour tout nouveau titre / nom à trouver, suivre ce protocole en 20 minutes.

```
Phase 1 — Ancrage (5 min)
□ À qui s'adresse ce contenu ?
□ Quel problème résout-il ?
□ Quel bénéfice principal en retire-t-on ?
□ En 1 mot : l'émotion qu'on veut provoquer ?

Phase 2 — Génération (10 min)
→ Écrire 10 variantes sans censure (voir AC.5 pour les 10 stratégies)
→ 2 variantes par type d'accroche de AC.2

Phase 3 — Sélection (5 min)
→ Scorer les 3 meilleurs avec la grille de mémorabilité (AC.7)
→ Lire à voix haute
→ Choisir 1 titular + 1 backup
```
